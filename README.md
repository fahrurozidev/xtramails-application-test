# xtramails-application-test

# BACKEND (Spring Boot)
- install JDK
- install Intellij
- install postgresql
- edit username and password postgresql in file xtramails-application-test/xtramiles-api/src/main/resources
/application.properties
- Run using intellij

# FRONTEND (Vue JS)
```sh
npm install
npm run dev
```
